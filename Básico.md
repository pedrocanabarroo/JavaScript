## 1. Operações Matemáticas
Este código mostra como trabalhar com números e realizar cálculos simples no JavaScript.

```javascript
// Variáveis para os números
const valorA = 20;
const valorB = 10;

// Cálculos
const soma = valorA + valorB;
const sub = valorA - valorB;
const mult = valorA * valorB;
const div = valorA / valorB;

// Exibição dos resultados
console.log("A soma é: " + soma);        // 30
console.log("A subtração é: " + sub);    // 10
console.log("A multiplicação é: " + mult); // 200
console.log("A divisão é: " + div);      // 2
